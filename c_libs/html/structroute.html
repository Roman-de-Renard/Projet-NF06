Detailed Description

structure représentant une route que peut prendre un vol

La structure route permet de représenter par sa fréquence, le nombre d'avion possible pouvant prendre cette route, la liste de ces avions, la ville de départ de la route, la ville d'arrivée, la capacité minimal pour prendre cette route ett la capacité maximal
